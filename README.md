# Chrome Dashboard

A simple Chrome extension that mimics the Okta End User Dashboard that displays names and urls of apps in sections. It uses chrome.storage.sync 
to store this data to one's Chrome profile so it persists in your account. It also users the Side Panel API to open the extension as part of the page
instead of the usual pop up effect.

Built using Opus 4.7. Prompt:

"Build a chrome extension using the side panel api that functions like the okta end user dashboard that allows the user to add the names and links of apps, divide them into sections, and move then around these sections. Store the data for sections, and the names and urls of links in chrome.storage.sync."
